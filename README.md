A program to send bulk emails

Usage -> 

```
$ python3 main.py --n <NUMBER-OF-FILES> --d <DELAY> --s <SUBJECT>
```

--n : Number of receivers file

--d : Delay between mails in secs

--s : Subject of mails

Format of receivers file->
  
```  
<sender_email>:<sender_app_pass>
<receiver_email>
<receiver_email>
.
.
.
```
